metrics-api

Scrapes public GitHub profile pages and aggregates npm registry APIs into clean JSON — no tokens, no auth.

Endpoint Returns
GET /github/:user/contributions per-day contribution counts + totals per year
GET /github/:user/profile name, bio, avatar, follower/following counts, orgs
GET /github/:user/repos public repos with stars, forks, language
GET /npm/:user packages with publish history + windowed daily downloads
